## ΜΕΡΟΣ Α: MI = Μέλι (Honey Determinative)

### A1. Γραμμική Β: me-ri = μέλι — ΕΠΙΒΕΒΑΙΩΜΕΝΟ

**Πηγή 1:** AJA Review (Melena, *Companion to Linear B* vol.3):
> «me-ri [honey] was not /melit/ as in Hittite» — τελικά σύμφωνα παραλείπονται στη Μυκηναϊκή ορθογραφία.

**Πηγή 2:** Palaima (UT Austin), *Mycenaean Ideograms*:
- Λογόγραμμα MERI = ME superimposed on RI = μέλι (honey)
- Λογόγραμμα MARI = MA superimposed on RE = μαλλί (wool)
- Οι γραφείς επέλεξαν RE αντί RI στο MARI «to clearly contradistinguish it from MERI»
- **Ακροφωνικό σύστημα:** «phonetic ideograms generally represent the first syllables of the words they designate»

**Πηγή 3:** Palaima, *Linear B Sources*:
- Ιδεογράμματα = «OIL, HONEY, GOLD CUP, MAN, SHEEP, WHEAT»
- NI = σύκα (από Κρητικό *nikuleon*) — **Μινωική προέλευση!**
- SA = λινάρι (ακροφωνική συντόμευση «pre-Hellenic word for linen»)

**Πηγή 4:** KN Gg 713 (Κνωσός):
- «to all the gods: honey: 1 jar» — θρησκευτική προσφορά
- Προσφορά μελιού στην «Πότνια του Λαβυρίνθου»

### A2. Γραμμική Α: MI ως Determinative Μελιού

**Αρχή Ακροφωνίας:**
Αν στη Γρ.Β το λογόγραμμα MERI χρησιμοποιεί το πρώτο συλλαβόγραμμα ME(RI) για μέλι, τότε στη Γρ.Α — όπου η ίδια αρχή ισχύει (NI = σύκα, SA = λινάρι) — το MI μπορεί να αντιπροσωπεύει το πρώτο συλλαβόγραμμα της Μινωικής λέξης για μέλι.

**Σημαντική παρατήρηση:** Η ίδια ρίζα *mel- (μέλι) είναι πανευρωπαϊκή: PIE *mélit > Ελλ. μέλι, Λατ. mel, Χεττ. milit, Γοτθ. miliþ. Η αλληλουχία MI- ταιριάζει τόσο με IE *mel- όσο και με Σημιτικό (π.χ. Αραβ. مَلَّ).

### A3. OLE+MI = Μελόλαδο (Honey-Infused Oil)

**Corpus attestation: 14 πινακίδες** — HT100, HT101, HT116a, HT125a, HT137, HT14, HT28a, HT42+59, HT50a, HT58, HT90, HT91, TY3a, TY3b.

Αν MI = μέλι:
- OLE+MI = ελαιόλαδο + μέλι = **μελόλαδο**
- §7A.5c Λειτουργική Αυτοτεκμηρίωση: OLE = ιδεόγραμμα ελαίου, MI = determinative μελιού
- k* = 0.90 (Gold Standard αν MI = μέλι αποδειχθεί)

**Αρχαιολογικό παράλληλο:** Μικτά μέλι + λάδι = κοινό φαρμακευτικό/αρωματικό προϊόν στο Αιγαίο.

### A4. U-DI-MI = Υδρόμελι (Mead) — HT117a

**Σημείωση:** Η Schoep ταξινομεί ως «type IV (people?)» — πιθανός κατάλογος ονομάτων. Ωστόσο:

Αν U-DI = ὑδρ- (νερό) + MI = μέλι: **U-DI-MI = ὑδρόμελι (mead)**

### A5. Αρχαιολογική τεκμηρίωση υδρομελιού στη Μινωική Κρήτη

**McGovern et al. (1997, U. Penn Biomolecular Archaeology):**
- Βιομοριακή ανάλυση κεραμικών κατάλοιπων από Μινωική Κρήτη (2700-1200 π.Χ.)
- «Archaeologists have detected wine, beer and **mead** in Crete, often mixed in the same vessels»
- Ίχνη μελιού + χημικοί δείκτες ζύμωσης (οξαλικό ασβέστιο + τρυγικό οξύ)

**Roscher (κλασικός φιλόλογος):**
- «Fermented honey (mead) preceded wine as an entheogen in the Aegean world»
- «On some **Minoan seals**, goddesses were represented with **bee faces** (compare Merope and Melissa)»

**Ελληνική παράδοση:**
- Ορφικοί: ο Κρόνος μέθυσε με μέλι αγριομέλισσας (= υδρόμελι)
- Αριστοτέλης: συζητά το υδρόμελι στα *Μετεωρολογικά*
- Διοσκουρίδης: συνταγές υδρομελιού + μελόμελου (κυδώνι-μέλι)
- «Hydromeli preceded wine in Greece for a long time»

**PIE ετυμολογία:** *médʰu = μέλι/μέθη → Αγγλ. mead, Σανσκρ. madhu, Ελλ. μέθυ

## ΜΕΡΟΣ Β: U-DI = ὑδ- (Water Root)

### B1. Ετυμολογία ὕδωρ

- PIE *wódr̥ (γεν. *wednós) → Proto-Ελληνικό *údōr → Ελλ. ὕδωρ
- **ΔΕΝ είναι Pre-Greek** (Beekes). Αμιγώς IE.
- Cognates: Λατ. unda, Σανσκρ. udán, Αγγλ. water, Γερμ. Wasser
- ὑδρία = ὕδωρ + -ία = «αγγείο νερού» (κανονικός σχηματισμός)

### B2. U-DI-RI-KI = ὑδρία; — PH7a

**Πινακίδα PH7a** (Φαιστός, MM II, ~1800 π.Χ. — αρχαιότερη πινακίδα!):

```
a.1  DI-PA-JA       ← επικεφαλίδα (= κατάλογος αγγείων τύπου DI-PA)
a.2  U-DI-RI-KI     ← εγγραφή 1
a.3  JA-RU-MA-NE    ← εγγραφή 2
a.4  JA-SI-DA-RA    ← εγγραφή 3
b.3  VINc L E       ← ιδεόγραμμα κρασιού + ποσότητα
```

Context = **κατάλογος αγγείων** κάτω από DI-PA-JA heading.

Αν DI-PA = δέπας (k*=0.88, Gold Standard ✓) και DI-PA-JA = «δεπάσια» (κύπελλα), τότε:
- U-DI-RI-KI = ενδεχόμενο **ὑδρία** (αγγείο νερού)
- Φωνητική: U-DI-RI-KI → */udrikki/ → ὑδρική; / ὑδρία;*

**Πρόβλημα:** ὕδωρ = PIE, ΟΧΙ Pre-Greek → η Μινωική λέξη δεν μπορεί να *προέρχεται* από ὕδωρ.

**Πιθανές ερμηνείες:**
1. **Μινωικό δάνειο → Ελληνική:** Η Μινωική ρίζα U-DI (νερό) δανείστηκε ή επηρέασε την Ελληνική
2. **Σύμπτωση ηχητική:** Η Μινωική ρίζα U-DI τυχαίνει να μοιάζει φωνητικά
3. **Κοινό υπόστρωμα:** PIE *wed- και Μινωικό U-DI μοιράζονται αρχαιότερη ρίζα
4. **Μυκηναϊκή αντιστοίχιση:** Οι Μυκηναίοι ερμήνευσαν U-DI- μέσω δικής τους ρίζας ὑδ-

**k* = 0.60** (αβέβαιο λόγω ετυμολογικής αμφισημίας)

### Κρίσιμο ερώτημα: MI = μέλι — πόσο σίγουροι μπορούμε να είμαστε;

**ΥΠΕΡ:**
1. Ακροφωνικό σύστημα Γρ.Β: MERI = me-ri = μέλι. Πρώτη συλλαβή = ME.
2. MI ≈ ME (εναλλαγή i/e τεκμηριωμένη στη D-series: DI→DE)
3. OLE+MI: 14 πινακίδες — μελόλαδο = πραγματικό προϊόν
4. U-DI-MI: υδρόμελι = αρχαιολογικά αποδεδειγμένο στη Μινωική Κρήτη
5. NI (σύκα) = Μινωική προέλευση — η αρχή ακροφωνίας ισχύει Cross-script
6. Μινωική μελισσοκομία: σφραγίδες με θεά-μέλισσα (Μέλισσα, Μερόπη)

**ΚΑΤΑ:**
1. MI μπορεί να σημαίνει κάτι εντελώς διαφορετικό στη Μινωική
2. Η αλληλουχία MI δεν εμφανίζεται ποτέ μόνη της ως ιδεόγραμμα (πάντα σε σύνθετα)
3. Schoep type IV (people?) — U-DI-MI μπορεί να είναι όνομα
4. Δεν υπάρχει «MI = μέλι» καταγεγραμμένο σε κανέναν κατάλογο ιδεογραμμάτων Γρ.Α

### Τελική εκτίμηση MI = μέλι: **k* = 0.75 (Probable)**

Ανεβαίνει σε 0.85 αν:
- Βρεθεί 3ο MI-σύνθετο σε context τροφίμου/ποτού
- Ή αν U-DI-MI αποκλειστεί ως όνομα (corpus analysis)
