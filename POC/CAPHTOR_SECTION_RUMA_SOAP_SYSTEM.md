## 1. Εισαγωγή — Η Υπόθεση

Η ανακάλυψη του συστήματος RU-MA ξεκίνησε από μια παρατήρηση στο suffix σύστημα:

```
OLE (λάδι) = ήδη ΥΓΡΟ
OLE + TU   = αντίφαση (υγρό από υγρό)
OLE + TA   = ΜΕΤΑΣΧΗΜΑΤΙΣΜΟΣ υγρού → στερεό

→ OLE+TA = στερεό από λάδι = ΣΑΠΟΥΝΙ
```

Στη συνέχεια, η corpus ανάλυση του RU-MA έδειξε ότι:

```
ῥύμμα / ῥύμα (Αρχαία Ελληνικά) =
  καθαριστική ουσία, αυτό που ξεπλένει
  ← ῥύπτω = καθαρίζω
  ← ῥύπος = βρωμιά

RU-MA = *ruma = Μινωική λέξη για
        καθαριστικό/ρύμμα
```

**Κατεύθυνση επιρροής:** Μινωική λέξη *ruma → Ελληνικό ῥύμα — όχι το αντίστροφο.

## 2. Το -TA Suffix και η Σαπωνοποίηση

### Θεμελιώδης Κανόνας

```
TU = εκχύλιση: ΣΤΕΡΕΟ → ΥΓΡΟ παράγωγο
TA = μετασχηματισμός: ΥΓΡΟ → ΣΤΕΡΕΟ/επεξεργασμένο

OLE (λάδι, υγρό) + TA = OLE+TA
Τι γίνεται όταν βράσεις λάδι με τέφρα:
  Λάδι + K₂CO₃ (τέφρα ξύλου)
  → σαπωνοποίηση → ΣΑΠΟΥΝΙ (στερεό)

OLE+TA = σαπούνι ελαιολάδου
```

**Αρχαιολογική επιβεβαίωση:**
- Σαπούνι από ελαιόλαδο: τεκμηριωμένο από 2800 π.Χ. (Σουμέριοι)
- Χετταίοι (~1500 π.Χ.): σαπούνι για τελετουργική κάθαρση
- Αίγυπτος: natron + λάδι = καθαριστικό

## 3. Πλήρης Κατάλογος RU-MA Compounds

```
RU-MA-TA-SE    1 instance
RI-RU-MA-TI    1 instance
JA-RU-MA-NE    1 instance
RI-RU-MA       1 instance
KU-RU-MA       1 instance  ← HT115b (πλάκα KU-TA)
SI-RU-MA-RI-TA₂ 1 instance
RU-MA          1 instance  ← bare (HT64)
RU-MA-TA       2 instances ← HT29, HT99b
```

### Δομή Compounds

```
ΠΥΡΗΝΑΣ: RU-MA = βάση καθαριστικού

PREFIX (τύπος βάσει συστατικού/παραγωγού):
  KU-  = KU τύπος (πλάκα KU-TA)
  RI-  = RI τύπος
  JA-  = JA τύπος
  SI-  = SI (ασφόδελος) τύπος

SUFFIX (επεξεργασία/κατάσταση):
  -TA  = στερεό (RU-MA-TA = στερεό σαπούνι)
  -TA-SE = στερεό + ποιοτικός qualifier
  -TI, -NE, -RI-TA₂ = άλλοι qualifiers
```

## 6. RU-MA vs RU-MA-TA

```
RU-MA (υγρό):
  = υγρό καθαριστικό / ρύμμα
  Μέτρηση: αριθμός μονάδων
  Χρήση: άμεση εφαρμογή, βραχυπρόθεσμη

RU-MA-TA (στερεό):
  = RU-MA + TA = στερεοποιημένο ρύμμα
  Μέτρηση: τεμάχια (μπάρες σαπουνιού)
  Χρήση: αποθήκευση, εμπόριο, εξαγωγή

Αναλογία:
  RU-MA    = υγρό σαπούνι (όπως σήμερα
             το υγρό καθαριστικό)
  RU-MA-TA = στερεό σαπούνι (μπάρα)
```

## 8. Μεθοδολογικά Διδάγματα

### 8.1 Αντίφαση στο Suffix = Σήμα Κατατεθέν

```
ΚΑΝΟΝΑΣ: Αν ΥΓΡΟ + TU = αντίφαση,
         τότε το suffix ΔΕΝ είναι TU
         αλλά TA (μετασχηματισμός)

OLE (υγρό) + TU = αδύνατο
OLE (υγρό) + TA = σαπωνοποίηση ✓

Η αντίφαση ΑΠΟΚΑΛΥΠΤΕΙ τη σωστή
ερμηνεία του suffix.
```

### 8.2 Η Συνταγή Κρύβεται στη Λίστα

```
ΚΑΝΟΝΑΣ: Λίστα με [Α + Β + Γ + ΑΠΟΤΕΛΕΣΜΑ]
         = ΣΥΝΤΑΓΗ ΠΑΡΑΓΩΓΗΣ

HT99b:
  SI+SE-NI + DA-SI-*118 + ZU-DU → RU-MA-TA

Τα τρία πρώτα = ΣΥΣΤΑΤΙΚΑ
Το τελευταίο  = ΑΠΟΤΕΛΕΣΜΑ
```

### 8.3 Ο Τύπος Αποκαλύπτει τον Παραγωγό

```
ΚΑΝΟΝΑΣ: PREFIX + RU-MA = τύπος παραγωγού
                          ή συνταγής

KU-RU-MA = συνταγή KU (= KU-TA)
RI-RU-MA = συνταγή RI-...
JA-RU-MA = συνταγή JA-...

Ο prefix = "brand" / παραγωγός
```
