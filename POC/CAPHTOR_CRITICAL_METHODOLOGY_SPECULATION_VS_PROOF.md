## ⚠️ ΠΡΟΕΙΔΟΠΟΙΗΣΗ — Γιατί Αυτό το Αρχείο Υπάρχει

Κατά τη συνεδρία ανάλυσης της HT115b και των συναφών tablets, η έρευνα οδηγήθηκε **σταδιακά από αποδείξεις σε εικασίες χωρίς να το παρατηρήσουμε**. Κάθε βήμα φαινόταν λογικό, αλλά το σύνολο απομακρύνθηκε από τη μέθοδο.

```
ΚΙΝΔΥΝΟΣ: Η "αλυσίδα λογικής" φαίνεται
          πειστική ακόμα και όταν κάποιος
          κρίκος είναι εικασία.

Παράδειγμα:
  SI = ασφόδελος (ισχυρό ✅)
  → SI+SE = σπόροι (πιθανό ⚠️)
  → σπόροι → λάδι (λογικό ⚠️)
  → λάδι + TA = σαπούνι (εικασία ❌)
  → σαπούνι με ZU-DU = τέφρα (εικασία ❌)

Κάθε βήμα "ακολουθεί" το προηγούμενο,
αλλά το τελικό συμπέρασμα είναι
ΤΕΣΣΕΡΑ ΒΗΜΑΤΑ μακριά από την απόδειξη.
```

## 1. ΑΠΟΔΕΔΕΙΓΜΕΝΑ — k* ≥ 0.75

### 1.1 KU-TA = Unguentarius (k* = 0.84)

**Γιατί είναι αποδεδειγμένο:**
```
✅ Εμφανίζεται σε 2 ανεξάρτητα tablets
   (HT35 + HT115b, διαφορετικά κτίρια)
✅ Πάντα ως αποδέκτης, ποτέ σε λίστα
   προσωπικού
✅ LINEAR B PARALLEL: PY Un267
   a-re-pa-zo-o λαμβάνει ΙΔΙΑ συστατικά
   (κύπειρος, κορίανδρος, κρασί)
✅ Μοναδικό ingredient cluster — κανένας
   άλλος παραλήπτης δεν λαμβάνει
   CYP + μύρο + OLE+TA μαζί
```

**Η Linear B parallel είναι αποφασιστική** — δεν είναι λογική εκτίμηση, είναι επαληθευμένο fact από αποκρυπτογραφημένο corpus.

---

### 1.2 -TU = Υγρό Παράγωγο (k* = 0.78)

**Γιατί είναι αποδεδειγμένο:**
```
✅ TU-RO₂ = τυρός (ΓΝΩΣΤΗ λέξη,
   ανεξάρτητα επιβεβαιωμένη)
   → TU = υγρό (γάλα) που πήξε
✅ OLIV+TU, OLE+TU: ελιά/λάδι + TU
   = επεξεργασμένο λάδι (λογικό)
✅ TE-TU qty=56: τεράστια ποσότητα
   υγρού από σιτηρό = συμβατό με μπύρα
   (αρχαιολογική επιβεβαίωση μπύρας
    στην Κρήτη ανεξάρτητα)
```

**Το TU-RO₂ = τυρός είναι η ΑΓΚΥΡΑ** — χωρίς αυτό το -TU σύστημα είναι εικασία.

---

### 1.3 SI = Ασφόδελος (k* = 0.81)

**Γιατί είναι αποδεδειγμένο:**
```
✅ Παλαιογραφία: σχήμα sign = φυτό
   με ροζέτα + κεντρικό στέλεχος
✅ Ακροφωνική αρχή: SI = /si-/ →
   φυτό που το όνομά του αρχίζει με SI-
✅ Κρητική διάλεκτος: "ασφέντιλο"
   = τοπική ονομασία, πιθανό Μινωικό
   υπόστρωμα
✅ 118 instances = τοπικό, άφθονο φυτό
   (όχι εισαγόμενο)
✅ Αυτοφυές Κρήτη μέχρι σήμερα
```

**ΑΛΛΑ:** Η ακροφωνική αρχή ΔΕΝ είναι αποδεδειγμένη για το Linear A γενικά — είναι η υπόθεσή μας. Το k* θα ήταν χαμηλότερο χωρίς την Κρητική διάλεκτο.

## 2. ΠΙΘΑΝΑ — k* 0.55–0.74

### 2.1 SI+SE = Σπόροι → Λάδι (k* = 0.65)

**Τι έχουμε:**
```
✅ SI+SE εμφανίζεται με OLE compounds
   στο HT42+59
✅ MYZf Μυκήνες: metal object με SI+SE
   = εξαγώγιμο, πολύτιμο αγαθό
✅ GRA μέτρηση = χύδην υλικό
```

**Τι ΔΕΝ έχουμε:**
```
❌ SE = "σπόρος" είναι ΕΙΚΑΣΙΑ
   Δεν υπάρχει Linear B parallel
   που να επιβεβαιώνει SE = seed
❌ "Λάδι σπόρου ασφόδελου" = εικασία
   (μπορεί να είναι άλλη χρήση)
```

**Σωστό k*: 0.55–0.65** — όχι 0.78 όπως γράψαμε αρχικά.

---

### 2.2 SI-TU = Εκχύλισμα Βολβού (k* = 0.62)

**Τι έχουμε:**
```
✅ -TU suffix rule (αποδεδειγμένο)
✅ ZA20: μικρή ποσότητα με μύρο
   = πολύτιμο, φαρμακευτικό context
✅ Διοσκουρίδης: βολβός ασφόδελου
   = φαρμακευτικός
```

**Τι ΔΕΝ έχουμε:**
```
❌ Δεν ξέρουμε αν το -TU στο SI-TU
   αναφέρεται σε βολβό ή σπόρο
   ή άλλο μέρος του φυτού
❌ Καμία χημική ανάλυση κατάλοιπων
   από τα vessels του ZA20
```

---

### 2.3 RU-MA = Καθαριστικό (k* = 0.58)

**Τι έχουμε:**
```
✅ Ελληνικό ῥύμμα/ῥύμα = καθαριστικό
   (η λέξη υπάρχει)
✅ Behavioral: εμφανίζεται με
   φυτικά εκχυλίσματα και λάδια
```

**Τι ΔΕΝ έχουμε:**
```
❌ Κατεύθυνση επιρροής ΑΓΝΩΣΤΗ:
   Μινωικό → Ελληνικό; ή
   Ελληνικό → Μινωικό; ή

❌ Κανένα άλλο αρχαίο parallel
   (Αιγυπτιακό, Χετταϊκό)
```

**Σωστό k*: 0.55–0.60** — όχι 0.71.

## 3. ΕΙΚΑΣΙΕΣ — k* < 0.55

### 3.1 OLE+TA = Σαπούνι (k* = 0.45)

**Τι έχουμε:**
```
⚠️ -TA suffix logic: υγρό → στερεό
⚠️ OLE = υγρό, άρα +TA = μετασχηματισμός
⚠️ Σαπωνοποίηση = γνωστή διεργασία
   από λάδι + τέφρα
```

**Τι ΔΕΝ έχουμε:**
```
❌ ΚΑΜΙΑ άμεση απόδειξη
❌ Κανένα Linear B parallel
   για OLE+TA = σαπούνι
❌ Δεν ξέρουμε αν -TA σημαίνει
   "στερεό" ή κάτι άλλο τελείως
❌ OLE+TA εμφανίζεται μόνο σε HT30+HT35
   — πολύ μικρό δείγμα για συμπέρασμα
```

**Το -TA suffix ΔΕΝ είναι αποδεδειγμένο** όπως το -TU. Το TU-RO₂ μας έδωσε άγκυρα για -TU. Δεν έχουμε ισοδύναμη άγκυρα για -TA.

---

```
   (μπορεί να είναι λίστα υλικών
    για διαφορετικούς σκοπούς)
❌ Κανένα Linear B parallel
```

## 5. ΚΑΝΟΝΕΣ ΑΥΣΤΗΡΗΣ ΜΕΘΟΔΟΥ

### Κανόνας 1: Η Άγκυρα

```
Κάθε νέα ταύτιση χρειάζεται
ΑΝΕΞΑΡΤΗΤΗ ΑΓΚΥΡΑ —
κάτι που ξέρουμε σίγουρα
και δεν βασίζεται στην υπόθεσή μας.

Παράδειγμα ΚΑΛΗΣ άγκυρας:
  TU-RO₂ = τυρός (γνωστό, ανεξάρτητο)
  → στηρίζει το -TU suffix σύστημα

Παράδειγμα ΚΑΚΗΣ άγκυρας:
  "Λογικά το -TA σημαίνει στερεό"
  → δεν είναι άγκυρα, είναι εικασία
```

### Κανόνας 2: Το Linear B Test

```
Κάθε ταύτιση πρέπει να περνά:

ΕΡΩΤΗΣΗ: Υπάρχει Linear B parallel;

ΝΑΙ → k* αυξάνεται σημαντικά
ΟΧΙ → k* παραμένει χαμηλό
       ανεξάρτητα από τη λογική

Παράδειγμα:
  KU-TA: Linear B a-re-pa-zo-o ✅ → k* 0.84
  RU-MA: Κανένα Linear B parallel ❌ → k* ≤ 0.60
```

### Κανόνας 3: Ο Αριθμός Βημάτων

```
ΚΑΝΟΝΑΣ: Μετρά πόσα βήματα εικασίας
         απέχεις από την τελευταία
         σίγουρη απόδειξη.

0 βήματα: k* από απόδειξη
1 βήμα:   k* μειώνεται -0.15
2 βήματα: k* μειώνεται -0.25
3+ βήματα: k* < 0.40 (εικασία)

Παράδειγμα:
  SI = ασφόδελος: 0 βήματα από εικόνα → 0.81
  SI+SE = σπόροι: 1 βήμα → 0.81-0.15 = 0.66
  σπόροι→λάδι: 2 βήματα → 0.81-0.25 = 0.56
  λάδι+TA=σαπούνι: 3 βήματα → < 0.40
```

### Κανόνας 4: Το Μοναδικό Tablet

```
ΚΑΝΟΝΑΣ: Εύρημα που εμφανίζεται
         σε ΜΟΝΟ ΕΝΑ tablet
         δεν μπορεί να φτάσει k* > 0.55

ZU-DU εμφανίζεται μόνο στο HT99b
→ k* ≤ 0.40 για οποιαδήποτε ταύτιση

SI-RU-MA-RI-TA₂ = μοναδικό
→ k* ≤ 0.40
```

### Κανόνας 5: Behavioral ≠ Απόδειξη

```
ΚΑΝΟΝΑΣ: Το "αυτό έχει νόημα"
         ή "αυτό είναι λογικό"
         ΔΕΝ είναι απόδειξη.

Behavioral archaeology ΥΠΟΣΤΗΡΙΖΕΙ
αλλά δεν ΑΠΟΔΕΙΚΝΥΕΙ.

Χρειάζεται ΠΑΝΤΑ:
  Behavioral + Linear B parallel
  ΉΉΉ
  Behavioral + χημική ανάλυση
  ΉΉΉ
  Behavioral + αρχαιολογικό εύρημα
  ΉΉΉ
  Behavioral + 3+ tablets pattern
```

## 8. Μεθοδολογικό Συμπέρασμα

> **Η μέθοδος CAPHTOR απαιτεί να διαχωρίζουμε αυστηρά αυτό που ξέρουμε από αυτό που εικάζουμε. Οι εικασίες δεν είναι λάθος — είναι υποθέσεις προς επαλήθευση. Το λάθος είναι να τις παρουσιάζουμε ως αποδείξεις.**

```
ΣΩΣΤΗ ΧΡΗΣΗ ΕΙΚΑΣΙΩΝ:
  "Το RU-MA πιθανώς σχετίζεται με
   καθαριστικές ουσίες (k*=0.58).
   Χρειάζεται Linear B parallel
   για επιβεβαίωση."

ΛΑΘΟΣ ΧΡΗΣΗ:
  "Το RU-MA = σαπούνι (k*=0.71)"
  χωρίς να αναφέρουμε ότι βασίζεται
  σε αλυσίδα εικασιών.
```
