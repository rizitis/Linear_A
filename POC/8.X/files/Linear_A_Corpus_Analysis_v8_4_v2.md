## 8b. ΜΕΘΟΔΟΛΟΓΙΚΗ ΑΡΧΗ — §7A.5c Λειτουργική Αυτοτεκμηρίωση [v2 — ΝΕΟ]

**Αρχή:** Όταν ένα ιδεόγραμμα εμφανίζεται **με κλάσμα** σε πινακίδα **χωρίς KU-RO/KI-RO** (= Level 1 Recipe κατά §7A.5b), η ταυτοποίησή του ως φυσικό προϊόν (φυτό, ρητίνη, λάδι) **τεκμηριώνεται αυτόματα από τη λειτουργία του στην πινακίδα**. Δεν χρειάζεται ετυμολογική απόδειξη.

**Αιτιολογία:** Ένα ιδεόγραμμα σε φαρμακευτική φόρμουλα με δοσολογία (κλάσμα) ΕΙΝΑΙ κατ' ανάγκη συστατικό. Η ετυμολογία (§16 TRAP) μας λέει ΤΙ ΟΝΟΜΑ ΕΙΧΕ — όχι ΤΙ ΕΙΝΑΙ. Το τι είναι φαίνεται από θέση, ποσότητα, γειτονικά συστατικά.

**Εφαρμογή στο CYP:** CYP + κλάσμα + χωρίς KU-RO σε HT35, KH12, KH85 → CYP = αρωματικό φυτό (κύπειρος). k\*=1.00 commodity, ανεξάρτητα ετυμολογίας.

**CYP — Τριστρωματική Ανάλυση (§16 TRAP + web search):**

| Layer | Αποτέλεσμα |
|-------|-----------|
| 0 — Γρ.Α | CYP λογόγραμμα, 100+ εμφ. ★ |
| 2 — Γρ.Β | ku-pa-ro₂ (KN Ga, PY Fr) ★ |
| 3 — Ελληνικά | κύπειρος = **Pre-Greek** (Μινωικό δάνειο). Η παλαιότερη μαρτυρία = Μυκηναϊκό ku-pa-ro |
| 4 — Σημιτική | Ακκαδ. **kapāru** «αλείφω» → **kupru** «πίσσα». Εβρ. **kōper** «χέννα». Ουγκαρ. **kpr** «άρωμα». Ρίζα K-P-R σημασιολογικά: αρωματική αλοιφή. **ΑΛΛΑ:** κύπειρος ≠ χέννα (2 φυτά) — Wanderwort hypothesis (k\*=0.55) |

**Τριστρωματικό k\*=0.92** (Gold Standard Tier A). Η Σημιτική σύνδεση K-P-R είναι **συμπληρωματική hypothesis**, ΟΧΙ θεμελιώδης.
