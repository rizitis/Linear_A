# CAPHTOR Protocol — Παράρτημα: Το Σύστημα SI (Ασφέντιλο)
## Πλήρης Ανάλυση Μινωικής Βιομηχανίας Ασφόδελου

**Ημερομηνία Ανακάλυψης:** Φεβρουάριος 2026  
**Corpus:** Linear A, Haghia Triada / Villa Magazine / Zakros / Μυκήνες  
**k* (SI = ασφόδελος):** 0.81  
**k* (SI+SE = λάδι σπόρου):** 0.78  
**k* (SI-TU = εκχύλισμα βολβού):** 0.74  
**Κατηγορία:** Commodity Analysis + Ακροφωνία + Suffix System

## 1. Εισαγωγή — Η Ανακάλυψη

Η ταύτιση του sign **SI** με τον **ασφόδελο** (Κρητική διάλεκτος: *ασφέντιλο / ασφεντιλιά*) έγινε μέσω τριών ανεξάρτητων μεθόδων:

1. **Παλαιογραφία:** Το σχήμα του sign απεικονίζει φυτό με ροζέτα φύλλων + κεντρικό ψηλό στέλεχος — χαρακτηριστική μορφολογία ασφόδελου
2. **Ακροφωνία:** SI = πρώτος ήχος της Μινωικής λέξης για το φυτό (*a-sfe-ti-lo → *si-...)
3. **Behavioral:** 118 instances = βασικό τοπικό αγαθό, όχι εισαγόμενο

Το φυτό **αυτοφύεται στην Κρήτη μέχρι σήμερα** και η τοπική του ονομασία "ασφέντιλο" διατηρεί Μινωικό υπόστρωμα (βλ. CAPHTOR_SECTION_ACROPHONY_LINEAR_A.md).

## 2. Τα Πέντε Compounds — Πλήρης Κατάλογος

### 2.1 SI (bare) — Ολόκληρο Φυτό

```
Χρήση: γενική αναφορά στο φυτό
Μέτρηση: αριθμός τεμαχίων
Instances: 118 (συχνότερο)
```

### 2.2 SI+SE — Σπόροι → Λάδι

**Tablets:**
| Tablet | Site | Context | Qty |
|--------|------|---------|-----|
| HT99b | Casa Room 7 | με RU-MA-TA | 6 |
| HT42+59 | Villa Magazine | αποκλειστικά OLE context | header |
| HT11? | Casa Room 9 | SI+SE GRA | 16 |
| MYZf1,2 | **Μυκήνες** | metal object (εξαγωγή!) | — |

**Ανάλυση:**
```
SI+SE εμφανίζεται ΠΑΝΤΑ με OLE compounds
  → SE = seed qualifier (σπόρος)
  → SI+SE = σπόροι ασφόδελου

Σπόροι → πίεση → ΛΑΔΙ ΣΠΟΡΟΥ ΑΣΦΟΔΕΛΟΥ

Μέτρηση με GRA (όγκος) = σπόροι σε χύδην
```

**Κρίσιμο εύρημα — MYZf Μυκήνες:**
```
Metal object (σφραγίδα/ετικέτα εξαγωγής)
από Μυκήνες με επιγραφή SI+SE

→ Λάδι ασφόδελου = εξαγώγιμο αγαθό
→ Αρκετά πολύτιμο για σήμανση σε μέταλλο
→ Εμπορική οδός: Κρήτη → Μυκήνες
```

**k* = 0.78**

### 2.3 SI-TU — Εκχύλισμα Βολβού

```
SI (φυτό, στερεό) + TU (υγρό suffix)
= υγρό παράγωγο βολβού ασφόδελου

Εφαρμογή -TU suffix κανόνα:
  ΣΤΕΡΕΟ + TU = ΥΓΡΟ ΠΑΡΑΓΩΓΟ
  βολβός ασφόδελου → εκχύλισμα
```

**Tablets:**
| Tablet | Site | Context | Qty |
|--------|------|---------|-----|
| ZA20 | Zakros (stone vessel) | με MI (μύρο), TE-AROM | 6 |
| HT3 | Portico 11 | SI-TU-RA₂-RE | 5 |

**Χρήση (Διοσκουρίδης, De Materia Medica):**
- Κατά αλωπεκίας (τριχόπτωση)
- Δερματικές παθήσεις
- Επουλωτικό πληγών
- Βάση αλοιφών

```
ZA20 context: qty=6 (μικρό, πολύτιμο)
Συν-εμφάνιση με μύρο = φαρμακευτικό/
καλλυντικό context
→ SI-TU = φαρμακευτικό εκχύλισμα
  για τον KU-TA (unguentarius)
```

**k* = 0.74**

### 2.4 DA-SI-*118 — Ξηρός/Τριμμένος Βολβός

```
DA + SI + *118
↑      ↑    ↑
proc. φυτό  μονάδα βάρους

DA = prefix επεξεργασίας (ξηρός/τριμμένος)
*118 = γνωστή μονάδα βάρους στο Linear A
```

**Tablets:**
| Tablet | Site | Context | Qty |
|--------|------|---------|-----|
| HT99b | Casa Room 7 | με SI+SE-NI, RU-MA-TA | 2 |
| HT13 | Villa Magazine | με TE-TU=56 (μπύρα!) | 19 |

**Κρίσιμο εύρημα — HT13:**
```
HT13 περιέχει:
  TE-TU      56  = μπύρα (τεράστια ποσότητα)
  DA-SI-*118 19  = ξηρός βολβός ασφόδελου

Βολβός ασφόδελου = ΑΜΥΛΟΥΧΟΣ
Αμυλούχα + ζύμωση = ζυμώσιμη ουσία

→ DA-SI = πρόσθετο στη ΖΥΜΩΣΗ μπύρας
  cf. Αρχαία Αίγυπτος: χουρμάδες/σύκα
      ως ζυμωτικά πρόσθετα
```

**Μέτρηση με βάρος (*118) vs αριθμό:**
```
DA-SI-*118 = μετριέται σε ΒΑΡΟΣ
           = υλικό τριμμένο/αποξηραμένο
           = δεν είναι ακέραια τεμάχια

Αντίθεση:
  SI+SE-NI qty=6 = 6 ακέραια τεμάχια/δέματα
  DA-SI-*118 = Χ μονάδες βάρους
```

### 2.5 SI+SE-NI — Σπόροι Τύπου NI

```
SI+SE (σπόροι) + NI qualifier
= συγκεκριμένος τύπος/ποιότητα σπόρου

Εμφανίζεται μόνο σε HT99b
qty=6 = μετριέται σε τεμάχια

NI = άγνωστος qualifier
     πιθανόν τόπος προέλευσης
     ή ποιότητα (1ης εκλογής;)
```

**k* (NI qualifier) = 0.42 (αβέβαιο)**

## 7. Μεθοδολογικά Διδάγματα

### 7.1 Ένα Φυτό — Πολλά Compounds

```
ΚΑΝΟΝΑΣ: Όταν ένα sign εμφανίζεται
         με πολλαπλά compounds,
         ψάξε για ΔΙΑΦΟΡΕΤΙΚΑ ΜΕΡΗ
         ή ΚΑΤΑΣΤΑΣΕΙΣ του ίδιου αγαθού

SI bare    = φυτό
SI+SE      = σπόροι (seed)
SI-TU      = υγρό (liquid extract)
DA-SI      = ξηρό (dried/processed)
SI+SE-NI   = συγκεκριμένη ποιότητα
```

### 7.2 Η Μέτρηση Αποκαλύπτει την Κατάσταση

```
ΚΑΝΟΝΑΣ: Ο τρόπος μέτρησης αποκαλύπτει
         τη φυσική κατάσταση του αγαθού

Αριθμός (1,2,6) = ακέραια τεμάχια
                  (φρέσκο, ολόκληρο)
GRA (όγκος)     = χύδην υλικό
                  (σπόροι, τριμμένο)
*118 (βάρος)    = επεξεργασμένο/ξηρό
                  (τριμμένο, αποξηραμένο)
```

### 7.3 Context = Χρήση

```
ΚΑΝΟΝΑΣ: Η συνεμφάνιση με άλλα αγαθά
         αποκαλύπτει τη χρήση

SI+SE με OLE    = λάδι (τρόφιμο/καλλυντικό)
SI-TU με MI     = φαρμακευτικό εκχύλισμα
DA-SI με TE-TU  = ζυμωτικό πρόσθετο (μπύρα)
SI με RU-MA-TA  = σαπούνι
```
